## Running the program

Run the makefile to generate the compilation objects and the final executable `tp`:

```bash
make
```

To run the program you must provide `{input_path}` and `{output_path}` with the flags `-i` and `-o`:

```bash
./bin/tp -i {input_path} -o {output_path}
```
